# Behavior Media and Datasets

Source IDs: `dryad_aedes_behavior_videos`, `mendeley_aedes_behavior_media`, `osf_flighttrackai_aedes_videos`, `pmc_open_access_videos`, `aedes_literature_facets`

This page covers Aedes aegypti behavior records, public video manifests, media locators, sound and wingbeat datasets, parsed behavior tables, and article supplementary media.

What it contains:

- 60,893 hosted behavior records
- Dryad Aedes behavior and video dataset manifests
- Mendeley Data behavior, media, high-speed video, wingbeat, hearing, and parsed table rows
- OSF FlightTrackAI processed and unprocessed Aedes flight-video manifests
- PMC open-access supplementary video locators
- behavior labels, file names, DOI or project locators, licenses, checksums, sizes, and download URLs when provided
