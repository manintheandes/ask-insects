# Neurobiology and Connectome Evidence

Source ID: `aedes_neurobiology_sources`

This page covers Aedes aegypti brain atlas, cell atlas, sensory neuron, CATMAID, voxel, SRA workflow, and connectome evidence.

What it contains:

- 100,018 hosted neurobiology records
- GEO GSE160740 processed brain snRNA-seq records
- SRA SRP290992 raw-access and reanalysis workflow metadata
- Mosquito Cell Atlas Zenodo files and H5AD internals
- MosquitoBrains volume headers, labels, and coordinate-queryable voxel access
- public Aedes EM/CATMAID metadata
- CATMAID skeleton export metadata and skeleton manifest records
- sensory-neuron and odor-encoding study metadata
- explicit source gap for a complete public whole-brain Aedes connectome bulk package
